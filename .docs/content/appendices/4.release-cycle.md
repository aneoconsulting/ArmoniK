# Release cycle

## A Rolling Release
- **Frequency:** Every 4-6 weeks
- **Features:** Contains all new features
- **Patches:** Security patches (CVE) and bug fixes on next release

---

## A Production Release
- **Frequency:** Every 6 months
- **Support Duration:** Supported for 9 months
- **Patches:** 1 Release every 4-6 weeks for security patches and bug fixes
